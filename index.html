What is Web Content Awareness ?

Web Content Awareness is a project developed by a 8th grade student - Krishna Sarathi Ghosh.

It is a Chrome extension which awares you of inappropriate images or content in a page, for example - Toxic and Offensive Content. This extension aims to make the web suitable for all ages!

Extension

Adding WebContentAwareness to Chrome

Download the Extension from here.

Download Extension (Click Here)

Open Chrome, click the three dot menu icon, click on Extensions, then click on Manage Extensions. Then turn on developer mode from the top right corner.

Finally,

Click on Load Unpacked and then Select the folder of the github repository.
